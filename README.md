# IT Support Portal

## Overview
IT Support Portal is a full-stack help desk ticketing system built with React, Spring Boot, and PostgreSQL. It allows users to create, view, filter, update, and manage support tickets through a multi-page frontend connected to a REST API backend. The project was built to simulate a real internal support workflow and demonstrate practical full-stack software engineering skills.

## Features
- Create support tickets
- View all tickets
- Search tickets by title or submitter
- Filter tickets by status, priority, and department
- Sort tickets by newest, oldest, and title
- View ticket details on a dedicated page
- Mark tickets as closed
- Delete tickets
- Dashboard summary with ticket counts
- PostgreSQL persistence
- Spring Boot REST API
- Backend validation
- Starter seed data for easier demo/testing
